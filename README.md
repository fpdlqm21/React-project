<h1>Market</h1>
<h2>1. 개요</h2>
React를 활용하여 SPA를 만들며 학습한다.
유저 간 중고물건을 사고 파는 중고거래 주제인 가상의 웹 어플리케이션으로 local server(json-server)통해 물건을 판매(추가), 구매(삭제) 그리고 검색 기능을 구현했다. 
<h2>2. 사용한 툴</h2>
React, Redux, Axios, Storybook, withStyles, redux-thunk 등을 이용하였다.
